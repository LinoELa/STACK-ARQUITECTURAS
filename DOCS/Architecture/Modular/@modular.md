# Carpeta Modular

Este archivo explica que representa la carpeta `Modular` dentro de `DOCS/Architecture`.

---

## 1. Que es esta carpeta

Esta carpeta agrupa toda la documentacion base para trabajar con Arquitectura Modular.

Aqui la idea principal es dividir el sistema en bloques funcionales claros, legibles y faciles de crecer.

---

## 2. Que contiene

Esta carpeta contiene:

- `@modular.md`
- `arquitectura-modular.md`
- `modular-rules.md`
- `dependency-rule.md`
- `audit-and-mapping.md`
- `migration-plan.md`

---

## 3. Como se lee

La lectura recomendada es esta:

1. overview general
2. reglas practicas
3. dependency rule
4. auditoria y mapeo
5. plan de migracion

---

## 4. Resumen rapido

La carpeta `Modular` sirve para documentar:

- que es la arquitectura
- como se organiza por bloques
- que reglas debe cumplir
- como auditar un proyecto
- como migrar hacia ella
